        \frametitle{C++ vs Python}
        \begin{itemize}
            \item C++:
            \begin{itemize}
                \item 編譯型語言（Compiled Language）
                \item 靜態型別（Static Typing）
                \item 高效能
            \end{itemize}
            \item Python:
            \begin{itemize}
                \item 直譯型語言（Interpreted Language）
                \item 動態型別（Dynamic Typing）
                \item 易學易用
            \end{itemize}
